HTML

HyperText Markup Language (HTML) is the standard language used to create web pages. Most of the web is based on this language.

HTML is made up of:
Text content
Markup (for edition)
References to other documents
Links

HTML Vocabulary

Tag: An HTML tag is always contained within angled brackets. Most tags have an opening tag <> and a closing one <> , but some just have an opening one <> They are called void tags.

Element: An HTML element refers to everything within a set of opening and closing tags.

Attribute: This is a property of an HTML element.
